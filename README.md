# FOCA (Fingerprinting Organizations with Collected Archives)

Custom build of FOCA that has the additional features
* **Delete button to delete projects
* **Bulk export user names and their document URLs to CSV
* **Longer waits between searches to lower the chances of getting rate limited by Google
